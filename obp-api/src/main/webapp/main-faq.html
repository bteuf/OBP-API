The base URL is

http://apisandbox.openbankproject.com
Please make sure you are using this in all your API calls

  1. Avoid using trailing slashes, else, you would get a 404 error. Example:


    .../obp/v1.4.0 200 OK
    .../obp/v1.4.0/ 404 Not Found
  2. Double check parameters are spelt correctly (including http vs https etc.)
  3. Check your encoding (use UTF8)

There are two ways to authenticate a user:
OAuth
and
Direct Login
.
If you are using this sandbox for a hackathon, we recommend you use Direct Login to authenticate as it is easier than the OAuth workflow.

If you want to use OBP with OAuth, we recommend you use (and fork) one of our
OAuth Starter SDKs.
If you are using this sandbox for a hackathon, we recommend you use
Direct Login.
